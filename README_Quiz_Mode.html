AI-Powered Personal Fact-Checker (Quiz Mode)

About This Project

This mini-project transforms a basic fact-checker into an interactive quiz application. It challenges users with general knowledge questions and immediately verifies their answers, providing instant feedback on correctness. The aim is to create an engaging and educational tool that demonstrates simple data handling and UI updates in a web environment.

How It Works

The core logic of this project is built using HTML, CSS (via Tailwind CSS), and JavaScript.

  1. Question Display: Upon loading, the application randomly selects a question from a predefined "encyclopedia" of questions (quizQuestions array in the JavaScript code). This question is then displayed in the designated "Question" box.
  2. User Input: The user types their answer into the "Your Answer" text area.
  3. Answer Verification: When the "Verify Answer" button is clicked (or Enter is pressed), the JavaScript code compares the user's input (after being trimmed and converted to lowercase for consistency) against the correctAnswer stored for the current question.
  4. Dynamic Feedback:
    • If the answer is correct, the "Verification Result" section updates to show "Correct!" with a green indicator, a justification, and 100% confidence.
    • If the answer is incorrect, it shows "Incorrect" with a red indicator, a justification, and low confidence.
    • If the answer box is left empty, it provides a prompt to enter an answer.
  5. Next Question: After displaying the result, the application automatically loads a new random question after a short delay (3 seconds), allowing for continuous quiz-taking.
  6. Styling: Tailwind CSS is used for responsive and modern styling, with specific colors applied to the background, question, answer boxes, and the verification button for a cohesive visual theme.

How to Run It

To run this project, follow these simple steps:

  1. Save the Code: Copy the entire HTML code provided in the ai-fact-checker-app Canvas.
  2. Create an HTML File: Paste the copied code into a new text file.
  3. Save the File: Save the file with an .html extension (e.g., quiz_fact_checker.html).
  4. Open in Browser: Open the saved HTML file using any web browser (e.g., Chrome, Firefox, Edge).

The quiz interface will load, presenting you with a random question. Type your answer and click "Verify Answer" to see the result and proceed to the next question.
